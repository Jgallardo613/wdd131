About AeroWeather

Welcome to **AeroWeather**! This dynamic web application was developed by **Jeffrey Gallardo** as a final project for the WDD 131: Dynamic Web Fundamentals course.

Our goal was to build a clean, functional, and **responsive** platform that accurately retrieves and displays current weather conditions and a 5-day forecast for any location worldwide using modern web standards.

Key Project Highlights:

  • **Dynamic Data:** Fetched using the JavaScript `fetch` method from the OpenWeatherMap API.
  • **Responsiveness:** Utilizes CSS Grid and Media Queries for optimal display on mobile and desktop devices.
  • **Data Handling:** Demonstrates proficiency in using JavaScript objects, arrays, and array methods (filter, forEach) to process the forecast data.
  • **User Experience:** Includes features like `localStorage` to remember the user's last search.
